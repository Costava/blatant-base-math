Base Math

Convert any number from any base to any base. Base 2 through 36 work out of the box. For higher bases, you'll need to define your own larger set of digits. Addition, subtraction, and multiplication are also supported between numbers of any bases.

Author: Costava | View source on GitHub

Example Input

Convert '1011' from base 2 to base 10.
convert('1011', 2, 10)
Add '1234' in base 10 with '12' in base 7. The result is in the base of the first number (base 10 in this example).
add('1234', 10, '12', 7)
Subtract '50' in base 10 from '11101' in base 2. The result is in the base of the first number (base 2 in this example).
subtract('11101', 2, '50', 10)
Multiply '80' in base 16 with '5' in base 10. You must specify the base of the answer in the last argument (base 2 in this example).
multiply('80', 16, '5', 10, 2)
